# 🐳 DevOps 

> Cahier de travaux pratiques — Docker · Prometheus · Grafana · Thanos

---

## 📚 À propos

Ce repo contient mes exercices pratiques pour apprendre les bases du **DevOps** et de l'**observabilité**.

L'objectif est de comprendre comment **conteneuriser** des applications et les **surveiller** en production.

---

## 🗂️ Structure du repo

### 🐳 Docker Basics
Découverte de Docker : comprendre la différence entre images et conteneurs, manipuler les commandes essentielles et appréhender la logique de conteneurisation.

### 🔥 Prometheus Basics
Introduction à l'observabilité : collecter des métriques avec Prometheus, les visualiser avec Grafana et assurer leur persistance sur le long terme avec Thanos.
